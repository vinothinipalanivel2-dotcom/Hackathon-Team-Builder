# 🚀 Hackathon Team Builder

A full-stack web application to create, join, search, and manage hackathon teams.

## 🔥 Features
- Create Teams
- Join Teams
- Delete Teams
- Search Teams
- Live UI updates
- Backend API (Express)

## 🛠 Tech Stack
- React (Frontend)
- Node.js + Express (Backend)
- Axios (API calls)
